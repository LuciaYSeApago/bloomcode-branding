BloomCode is a collection of small C++ projects designed to strengthen my programming skills while embracing a cozy and creative aesthetic.


The goal isn't only to learn modern C++, but also to show that technical projects can have personality. Over time, many programming applications have evolved into a more minimalistic and utilitarian design. While that simplicity is often useful, I believe there's still room for playful interfaces, thoughtful design, and a little creativity.

Every project in this collection follows the same philosophy:

- 🌸 Cozy and welcoming
- 💖 Cute without sacrificing quality
- ✨ Clean and readable
- 🌱 Built to learn step by step
- 🎮 Fun to create and explore
- 💻 Written with modern C++

## Why BloomCode?

Learning to code often feels intimidating, especially when the beginner projects look overly technical and soulless.

BloomCode aims to make that first contact a little bit friendlier making these projects easy to understand, enjoyable to explore, and approachable enough that anyone can recreate them while learning.

## Representation matters

Technology was built by lots of people with many different backgrounds, interests and aesthetics.

Not everyone enjoys pastel colors or cozy interfaces, and that's perfectly fine.

BloomCode simply embraces a side of coding that is often ignored in programming spaces: a soft, playful and feminine visual identity.

My personal challenge is that anyone who enjoys this style feels a little more represented.

Programming is for everyone, and BloomCode celebrates the idea that technical skills and creativity—including traditionally feminine aesthetics—can absolutely coexist.

## Project Structure
Every BloomCode project follows the same design language, folder structure and documentation style to create a consistent learning experience across the entire collection.
