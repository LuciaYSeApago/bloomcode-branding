# 🌱 Philosophy

Icons are not decoration.

Each symbol has a meaning and should help users identify information quickly while keeping every BloomCode project visually consistent.

Whenever possible:

- Keep interfaces simple.
- Reuse the same symbols.
- Don't mix multiple aesthetics in the same project.
- Prioritize readability over decoration.
